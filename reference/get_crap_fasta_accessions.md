# Extract the contaminants protein accessions from a cRAP fasta file

Extract the contaminants protein accessions from a cRAP fasta file

## Usage

``` r
get_crap_fasta_accessions(cont_fasta_inf)
```

## Arguments

- cont_fasta_inf:

  `character` Filepath to the contaminants fasta

## Value

Returns a `list` of contaminant IDs
