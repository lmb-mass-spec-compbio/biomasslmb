# Benjamini & Bogomolov (2014) selective-inference adjustment using external screening p-values (e.g., F-test p-values)

Benjamini & Bogomolov (2014) selective-inference adjustment using
external screening p-values (e.g., F-test p-values)

## Usage

``` r
bb2014(mtx_p, p_screen, alpha = 0.05)
```

## Arguments

- mtx_p:

  Matrix of raw p-values (families Ã— hypotheses)

- p_screen:

  Vector of screening p-values (one per family)

- alpha:

  Target FDR level across selected families

## Value

A list with:

- adj_p: matrix of adjusted p-values (NA for unselected families)

- selected: integer indices of selected families

- alpha_star: the B&B reduced alpha
