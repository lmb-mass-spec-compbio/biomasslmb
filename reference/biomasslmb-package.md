# biomasslmb: BioMass LMB Proteomics Data Analysis Utility Functions

Utility functions for piping together quantitative proteomics data
analysis. Designed for projects conducted with the Biological Mass
Spectrometry and Proteomics Facility at the MRC Laboratory of Molecular
Biology.
