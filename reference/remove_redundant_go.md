# Remove redundant GO terms

Given the output of
[`get_enriched_go`](https://lmb-mass-spec-compbio.github.io/biomasslmb/reference/get_enriched_go.md)
or `goseq`, remove redundant GO terms.

## Usage

``` r
remove_redundant_go(
  obj,
  go_category_col = "category",
  p_value_col = "over_represented_pvalue"
)
```

## Arguments

- obj:

  `data.frame` containing `goseq` results as generated by

- \`go_category_col\`:

  `character` column name for GO terms

- \`p_value_col\`:

  `character` column name for p-values
  [`get_enriched_go`](https://lmb-mass-spec-compbio.github.io/biomasslmb/reference/get_enriched_go.md)
  or `goseq`.

## Value

Returns `obj` with redundant GO terms filtered out.
